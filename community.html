Community

Since the release of EtherPad as free software, a vibrant community has grown. Partake in the community driven version of EtherPad by using any communication channel below.

Source Code

All the source code for the free software release of EtherPad is available on GitHub. Please fork!

http://github.com/ether/pad

Mailing Lists

There are two main lists for EtherPad.

etherpad-open-source-discuss — user focused mailing list

etherpad-dev — developer focused mailing list

IRC

There is also an IRC channel which you can lurk on. At the moment it is a bit developer centric.

#etherpad@freenode
